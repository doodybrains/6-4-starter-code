Welcome to Intermediate CSS: Responsive Design & Flexbox!

Today we are going to cover several high level CSS concepts that are really helpful when making websites, especially if you want to make something based off a pre-designed layout. This could be something that you designed yourself or maybe something someone else designed. Responsive web design means that the website is built to flexibly respond to different screen sizes and devices.
